Box-Sizing Comparison

Left uses default content-box. Right uses border-box. Both have same width and padding.

content-box

Width: 300px; padding: 30px; border: 6px solid #cbd5e1

border-box

Width: 300px; padding: 30px; border: 6px solid #a7f3d0

When using border-box, padding and border are included in the specified width; with content-box, they are added outside the width.
