Responsive Design Principles

Fluid Layouts

Layouts that adapt using percentages rather than fixed units.

width: 80%;

Flexible Media

Images and videos scale with the container.

img { max-width: 100%; height: auto; }

Breakpoints

Using media queries to adjust design for different screen widths.

@media (max-width: 600px) { ... }
